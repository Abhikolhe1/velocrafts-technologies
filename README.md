# Velocrafts Technologies

Full-stack project for Velocrafts Technologies with React frontend and LoopBack 4 API backend.

## Structure

```
velocrafts-technologies/
├── velocraft-frontend/   # React + Vite + Tailwind frontend
├── velocraft-backend/    # LoopBack 4 REST API
└── README.md
```

## Quick Start

### Frontend

```bash
cd velocraft-frontend
npm install
npm run dev
```

Runs at http://localhost:5173

### Backend

```bash
cd velocraft-backend
npm install
npm start
```

- API: http://localhost:3000
- Explorer: http://localhost:3000/explorer
- Ping: http://localhost:3000/ping

## Build

```bash
# Frontend
cd velocraft-frontend && npm run build

# Backend
cd velocraft-backend && npm run build
```
